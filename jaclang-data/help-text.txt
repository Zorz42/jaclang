Jaclang help: 
usage:
   jaclang - for help
   jaclang [option] - for misc
   jaclang [input file] [output file] - for compilation

options:
   -d - debug - get more detailed compilation (for nerds)
   -k - keep  - keep cache (asm, object file located in .jlcache)
   -q - quiet - say nothing except errors and debug messages if specified

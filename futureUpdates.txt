BETA 1.2.1
  - jpm: Jaclang Package Manager
     -> it will come default with jaclang
     -> it will be written in python
     -> options:
     	install - installs library
	remove  - removes library
	list    - lists all installed libraries
	search  - searches for libraries in https://jaclang.zorz.si/repository
	clean   - cleans all unneeded dependencies
BETA 1.2.2
 - you can set variable after declaring it
 - output format recognition:
 	binary executable -> full compilation
	object file -> skips linker
	asm file -> skips linker and nasm
	jl file -> only preprocesses it

Jaclang help: 
!!!IT IS NOT RECOMMENDED TO RUN JACLANG ON IT'S OWN!!!
!!!IF YOU WANT TO COMPILE .jl FILES, USE JACMAKE!!!
usage:
   jaclang - for help
   jaclang [input file] - for compilation

options:
   --debug-*          - get more detailed compilation (for nerds):
      --debug-ast       - show ast
      --debug-tokens    - show tokens
      --debug           - show ast and tokens
   --quiet            - say nothing except errors and debug messages if specified
   --version          - display version
   --no-optimizations - do not optimize code (for debugging)

parameters:
   -o [output file] - set output file
